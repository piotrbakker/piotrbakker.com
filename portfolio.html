Videoshop

Freelace Product Designer

Mobile app design, User Interface Design · 2023

Most recently I’ve redesigned a video editing app for iOS called Videoshop. The main challenge was to create a unified user interface for over fifty different tools and features, for all iOS and iPadOS devices and screen sizes, including support for Dark Mode.

Wizehire.com

Freelace Product Designer

Design System, Web Design, User Interface Design · 2020–2022

I helped WizeHire establish a formal design system to maintain design consistency across all of the company’s products and allow for more effective product development. I’ve streamlined the use of typography, color, layout grid, and set up a component library on Figma.

Galactic Escape

Maker

Art Direction, Graphic Design, Game Design · 2016

Galactic Escape was a retro side-scrolling game for iOS and tvOS. I created the original concept for the game, set out the art direction, prepared image asset files, organized and lead the team of 3 software engineers. The game was launched on the tvOS App Store shortly after the debut of the 4th generation Apple TV and was subsequently ported to iOS as well.

Mixbook.com

Freelance Product Designer

Design System, Responsive Web Design, User Interface Design · 2016–2019

Mixbook asked me for help with redesigning their website using responsive web design principles.  I worked with the internal design and marketing teams to establish a new visual identity and design system, as well as to update key web pages: the photo book editor, storefront and checkout.
